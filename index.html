1-ая фаза
 
1-ый день
Утро Чай зеленый + започенное яблоко (1-2 шт) не сладкий сорт, по гототовности присыпать укропом
Обед Суп на рыбном бульоне: шампиньоны 2-3 шт. + картофель 1 + стручковая фаслоь (заморозка) + укроп + шпинат. Через 2-3- мин. вода или чай зеленый (если хочется)
16:00 -
Ужин Отварная рыба + присыпать укропом. ПОСЛЕ ничего не едим и не пьем
2-ой день
Утро Кофе + клубника
Обед Варим в слегка подсоленой воде зеленую гречку + в конце доварим + укропом + шпинат. Чай зеленый
16:00 Кофе + клубника
Ужин Чай зеленый + запеченые яблоки (1-2 шт.)
3-ий день
Утро Кофе + малина
Обед Кабачок (он же цукини) - запекаем с оливковым маслом + сок лимона сбрызнуть. Чай зелёный
16:00 Чай зеленый + яйцо вареное
Ужин Как обед + чай зеленый
4-ый день
Утро Коктейль: кефир + киви + укроп + клубника
Обед На рыбном бульоне варим суп: цветная капуста + морковь + сельдерей + шпинат + укроп + допустимо часть от рыбы. Через 15-20 мин чай зеленый или вода.
16:00 -
Ужин Как завтрак
5-ый день
Утро Кофе + апельсин
Обед Морская капуста + укроп + шпинат. Чай зеленый
16:00 -
Ужин Отварная рыба (от 4 дня) + присыпать укропом. Чай зеленый
6-ой день
Утро Кофе + киви (1 2 шт)
Обед апеченная красная рыба (лосось) + укропом спежим присылать. Чай зеленый.
16:00 -
Ужин Варим зеленую гречку + допарить п конце укроп + шпинат. Напиток на выбор.
!!! ОБЯЗАТЕЛЬНО УТОРМ 7-го ЖДУ ОЧТЕТ ПО ВЕСУ!!! (В случае необходимости вношу корректировки)
7-ой день
Утро Кофе + апельсин
Обед На рыбном бульоне суп: нерка/лосось небольшой кусочек + сельдерей 1/3 + перец зеленый + стручковая фасоль +укроп + шпинат. Через 15 мин по потребности чай зеленый или вода.
16:00 Завтрак повторить
Ужин Шампиньоны + сбрызнуть соком лимона + оливковое масло + укроп = запечь. Напиток на выбор.
 
2-ая фаза
8-ой день
Утро Кофе + малина
Обед Куриная грудка вареная + присыпать укропом. Через 30 мин. яблоко зеленое. Чай зеленый.
16:00 Завтрак
Ужин Кабачок запечь с тертым яблоком + присыпать укропом
9-ый день
Утро Кофе + ежевика
Обед Суп ня бульоне от грудки: брокколи + картофель + морковь + помидор + шпинат.
Ужин Яйцо вкрутую + вишня.
10-ый день
Утро Кофе - вишня (100-150 гр.).
Обед Вареный рис (черный или бурый) + в конце доварить укроп + шпинат + горсть брусники. Чай зелёный
16:00 Кофе + яблоко зеленое
Ужин Чай + запеченное яблоко (1-2)
11-ый день
Утро Кофе черный + вишня
Обед Запекаем кабачок + шпинат + сыр тертый. На десерт яблоко + чай зеленый.
16:00 -
Ужин Куриная грудка + вишня. Напиток на выбор.
12-ый день
Утро Коктейль: кефир + 1 киви - 3-4 клубника
Обед Варим цветную капусту + слегка отбланшировать (значит потушить немного под крышкой с водой) на оливковом масле с соком лимона + присыпать укропом и тертым сыром. Чай зеленый / вода.
Ужин Завтрак
13-ый день
Утро Кофе + смузи: 1 2 зеленое яблоко + горсть грецкий орех + укроп + 50 мл вода
Обед Бульон от курин грудки + 1 ст. ложка зеленого горошка (не консерва) + яйцо вкрутую + укроп (суш или как угодно). На десерт можно яблоко зеленое. Через 30 мин по потребности чай зеленый /вода.
16:00 Завтрак
Ужин Не более 30-50 грамм сыр (не более 25% жирности) + зеленое яблоко + чай зеленый или вода.
!!! ОБЯЗАТЕЛЬНО УТОРМ 14-го ЖДУ ОЧТЕТ ПО ВЕСУ!!! (В случае необходимости вношу корректировки)
14-ый день
Утро Чашка зелёного чая + яйцо варёное вкрутую (1-2 шт.).
Обед Стейк: нерка/форель (красная любая рыба по желанию) запечённая в духовке в фольге + сок лимона вместо соли + укроп + шпинат. Зеленое яблоко без кожуры 1 шт. Через 15 мин. Чай зелёный.
16:00 Кофе + малина (заморозка) 100-150 гр.
Ужин Шампиньоны свежие, немного тушёные в сковороде под крышкой с укропом + шпинат. Чашка зеленого чая.
3-я фаза
15-ый день
Утро Кофе + киви 1-2 шт.
Обед Творог (не более 2% жирности и в составе не должно быть сухого молока) + укроп + шпинат + 1 ст. ложка сметаны 15% + огурец.
Ужин Коктейль = 100-150 мл. воды + киви + яблоко зеленое.
16-ый день
Утро 2 яблока + кофе черный.
Обед Варим (ЗЕЛЕНУЮ) гречку + укроп + горсть опят (заморозка) + шпинат (доварить).
Ужин Смузи хорошо взбить: 100 -150 мл. воды + киви + зеленое яблоко + сельдерей + укроп + шпинат.
17-ый день
Утро Яблоко зеленое (1-2 шт.) + кофе черный.
Обед Творог (не более 2% жирности и в составе не должно быть сухого молока) + укроп + шпинат + руккола + яблоко зеленое. Соблюдаем размер порции.
Ужин Коктейль = 100-150 мл. воды + клубника + зеленое яблоко + сельдерей + укроп + шпинат.
18-ый день
Утро Гранатовый сок + вестфальский хлеб или бездрожженой хлеб темных сортов (25-50 гр.).
Обед Варим суп на нежирном бульоне из указанных ингредиентов: шампиньоны, морковь, свекла, помидор розовый, укроп, шпинат. Количество ингредиентов определяем «на глаз». Порция готового супа не должна превышать 200-250 гр. Мясо, рыбу от бульона не едим.
Ужин Гранатовый сок (120-150 мл.) + вестфальский хлеб (25-50 гр.).
19-ый день
Утро Кофе - ананас (100-150 гр.).
Обед Тунец (натуральный или консервы в собственном соку, едим целую или половину баночки) + укроп + руккола + чай зеленый.
Ужин Творог (не более 2% жирности и в составе не должно быть сухого молока) + ягода любая.
20-ый день
Утро Кофе + 1-2 шт. зеленого яблока.
Обед Грудка (все кроме жарки) + укроп + шпинат + яблоко зеленое.
Ужин Смузи/шойк/просто блсидср.нуть: ½ сельдерея + 2-3 ветки укропа + 1 зеленое яблоко + 100-150 мл. воды негазированной + горсть клубники (заморозка) + 2-3 листика шпината. Съесть/выпить. Больше ничего не едим, не пьём.
!!! ОБЯЗАТЕЛЬНО УТОРМ 21-го ЖДУ ОЧТЕТ ПО ВЕСУ!!! (В случае необходимости вношу корректировки)
21-ый день
Утро Кофе + клубника (100-150 гр.).
Обед Тушим капусту белокочанную + брусника + шпинат + укроп затушить. Не пьём. Через 30-60 мин. кофе с клубникой.
Ужин Коктейль = 100-150 мл. воды + укроп, сельдерей, яблоко (без кожуры), клубника. 150-170 гр. запечённой или вареной телятины/говядины.
4-ая фаза
22-ой день
Утро Кофе + клубника (100-150 гр.).
Обед Грудка (все кроме жарки) + брусника + шпинат. Зелёный чай.
Ужин Коктейль = 100-150 мл. воды + укроп + шпинат + сельдерей + брусника (80-100 гр.). 1-2 яйцо варёное вкрутую.
23-ий день
Утро Кофе + гранат 1 шт. (Едим вместе с косточками).
Обед Варим суп на нежирном бульоне из указанных ингредиентов: шампиньоны, красный перец, шпинат, укроп. Количество ингредиентов определяем «на глаз». Порция готового супа не должна превышать 200-250 гр. Мясо, рыбу от бульона не едим.
Ужин Гранат (зерна) едим вместе с косточками + 50-80 гр. сыра Качотта. Не запивать. Через 30-40 мин. чашка гранатовой сока 120-150 мл.
24-ый день
Утро Кофе + клубника (100-150 гр.).
Обед Тушим кабачок + шпинат + укроп + петрушка + брусника (горсть).
16:00 Кофе + клубника (100-150 гр.).
Ужин Как обед + 2 огурца.
25-ый день
Утро Кофе - вишня (100-150 гр.).
Обед 150 гр. телятины/говядины + укроп + шпинат.
Ужин Кофе + вишня (100-150 гр.).
26-ой день
Утро Кофе + облепиха (100-150 гр.).
Обед Шампиньоны + шпинат + креветка (6-10 шт.) + облепиховый сок (120-150 мл.).
Ужин Кофе + зеленое яблоко.
!!! ОБЯЗАТЕЛЬНО УТОРМ 27-го ЖДУ ОЧТЕТ ПО ВЕСУ!!! (В случае необходимости вношу корректировки)
27-ой день
Утро Кофе + запеченное яблоко (1-2 шт.).
Обед Тунец (натуральный или консервы в собственном соку, едим целую или половину баночки), укроп, руккола.
16:00 Кофе + 1 зеленое яблоко.
Ужин Тунец (натуральный или консервы в собственном соку, едим целую или половину баночки), укроп, руккола, зеленое яблоко, огурец.
28-ой день
Утро Кофе + клубника (100-150 гр.).
Обед Тушеные шампиньоны (СВЕЖИЕ готовим не заморозку) + шпинат. Через 15 мин. кофе + клубника (80-100 гр.).
Ужин Клубника (80-100 гр.) + хлеб вестфальский (25-50 гр.).
